Reporter View Companion



The goal of this project is to bring all of the data gathered by Reporter for reflection and analysis.

To see an example of what this does visit sambev.com.

Contributions are deeply appreciated. Fork the project on GitHub.

As you might have noticed, I am no designer. Any design contributions are more than welcome as well. If you have some improvements, but don't code, just contact me.


***DISCLAIMER***

Before creating an account and uploading data, understand that this is a passion project supported by myself and done in my spare time. While I try to make security conscious decisions, I simply cannot gaurantee the safety of your data uploaded through this domain. If you have noticed a security issue, please contact me.

UPLOAD AT YOUR OWN RISK

If you have sensitive data but would still want to use the product. You can always download the code from GitHub, follow the install instructions and run the application locally on your machine. If you have problems getting it running please use the issue tracking page and I'll try to help where I can.
